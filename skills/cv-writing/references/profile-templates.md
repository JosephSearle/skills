# Profile / Personal Statement Templates

## Structure

Every profile follows four beats in order:

1. **Identity** — years of experience + specialism + key stack (1 sentence)
2. **Proof** — 1–2 flagship achievements with concrete numbers (1–2 sentences)
3. **Stack signal** — exact frameworks the target company uses (1 sentence)
4. **Aim** — what you want to do next, mirroring the JD's first 3 responsibilities or mission (1 sentence)

**Length**: 50–80 words, **first-person implicit** (no "I"). UK English throughout.
Do not open with "I am" — omit the pronoun and start with the identity descriptor.

---

## How to Tailor

Before writing the profile:
1. Extract the JD's first 3 responsibilities and mission statement (from jd-parser.md output)
2. Identify which 3–4 frameworks from keyword-vocabulary.md the target company names explicitly
3. Pick the candidate's top 2 achievements that best map to those responsibilities
4. Drop those frameworks and achievements into the template below

## Anti-Patterns

| Weak profile | Why it fails |
|-------------|-------------|
| "Passionate software engineer looking for new challenges in AI" | No experience, no stack, no numbers, no aim |
| "Experienced in machine learning, NLP, computer vision, and data engineering" | Breadth without depth; no production signal |
| "I am a dedicated professional with strong communication skills" | First person, soft skills only, no technical content |
| "Seeking a challenging and rewarding opportunity in a dynamic organisation" | Generic objective — could apply to any candidate |
| "5 years of experience in various AI technologies including Python and TensorFlow" | Vague stack, no achievements, no company-type targeting |
