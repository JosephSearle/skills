# @carbon/ibm-products Setup Reference

> Authority: [github.com/carbon-design-system/ibm-products](https://github.com/carbon-design-system/ibm-products) and [carbondesignsystem.com/community/patterns/ai-label](https://carbondesignsystem.com/community/patterns/ai-label/)

`@carbon/ibm-products` extends `@carbon/react` with AI-specific and enterprise components. It is a separate package maintained by IBM.

---

## Installation

```bash
npm install @carbon/ibm-products @carbon/ibm-products-styles
```

Both packages are required: `@carbon/ibm-products` provides the components; `@carbon/ibm-products-styles` provides the CSS.

Add styles to `app/globals.scss` after the Carbon barrel import:

```scss
@use '@carbon/react';
@use '@carbon/ibm-products/css/index.min.css';
```

## AILabel component

`AILabel` decorates any Carbon component or container with an AI origin badge. It signals to users that content was generated by AI.

```tsx
'use client'
import { AILabel, AILabelContent } from '@carbon/ibm-products'
import { Tile } from '@carbon/react'

// On an AI-generated message bubble
<Tile>
  <AILabel size="xs" aiText="AI" />
  <p>{message.content}</p>
</Tile>

// On a form field pre-filled by AI
<TextInput
  id="suggestion"
  labelText="AI suggestion"
  value={aiSuggestion}
  slug={<AILabel size="mini" />}
/>
```

| Prop | Values | Notes |
|------|--------|-------|
| `size` | `'mini'` \| `'xs'` \| `'sm'` \| `'md'` \| `'lg'` \| `'xl'` | `'xs'` suits inline message bubbles |
| `aiText` | string | Defaults to `'AI'` |
| `kind` | `'default'` \| `'inline'` | `'inline'` for compact placement |

## Slug component

`Slug` is an alternative to `AILabel` that provides an icon-only AI indicator, typically placed inside other Carbon components via their `slug` prop:

```tsx
'use client'
import { Slug } from '@carbon/ibm-products'
import { DataTable } from '@carbon/react'

// AI-generated table rows
<DataTable
  rows={rows}
  headers={headers}
  slug={<Slug />}
/>
```

Most Carbon v11 components that accept AI annotation have a `slug` prop that accepts `<Slug />`.

## Available AI components (selection)

| Component | Use case |
|-----------|----------|
| `AILabel` | Badge on AI-generated content, available in multiple sizes |
| `Slug` | Compact icon-only AI indicator, accepted via `slug` prop on Carbon components |
| `CoachmarkOverlayElements` | Step-by-step AI feature introduction |
| `StatusIcon` | Status indicator with AI-appropriate iconography |

Check [ibm-products Storybook](https://ibm.github.io/ibm-products/) for the full component catalogue.

## Version alignment

`@carbon/ibm-products` must match the major version of `@carbon/react` in use:

| `@carbon/react` | `@carbon/ibm-products` |
|-----------------|----------------------|
| ^11.x | ^2.x |

WRONG — version mismatch:
```json
{
  "@carbon/react": "^11.50.0",
  "@carbon/ibm-products": "^1.x.x"
}
```

CORRECT:
```json
{
  "@carbon/react": "^11.50.0",
  "@carbon/ibm-products": "^2.x.x"
}
```
